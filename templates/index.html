Linux MacOS Windows Android
  1. 1. Install wireguard here
  2. 2. Generate your config here, and paste it to /etc/wireguard/wg0.conf
  3. 3. Activate the tunnel: sudo wg-quick up wg0
  4. 4. Test connectivity: ping dns.scinet.ctf
  5. 5. To close the tunnel use: sudo wg-quick down wg0

MacOS Info

  1. 1. Install wireguard here
  2. 2. Generate your config here
  3. 3. Open the wireguard app and press CTRL + N to add a new tunnel
  4. 4. Specify a name, paste your generated config and click Save
  5. 5. Select your newly created tunnel and click Activate
  6. 6. Test connectivity: ping dns.scinet.ctf
  7. 7. To close the tunnel re-open the wireguard app, select your connection and click Deactivate

Note: Although you can access the VPN from a mobile device, we strongly discourage it.
